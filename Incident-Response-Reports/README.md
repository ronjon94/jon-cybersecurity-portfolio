# Incident Response Reports

This section contains my incident response investigations, written using professional SOC and DFIR standards.  
Each report includes:

- Executive summary  
- Indicators of compromise (IOCs)  
- Attack timeline  
- Tools used (Splunk, Wazuh, Wireshark, Sysinternals, etc.)  
- Evidence and screenshots  
- Root cause analysis  
- Containment, eradication, and recovery steps  
- MITRE ATT&CK mapping  

These reports demonstrate my ability to analyze security events, document findings, and communicate clearly with technical and non‑technical audiences.
