# Threat Hunting

This section contains proactive threat hunts based on hypotheses, log analysis, and MITRE ATT&CK techniques.

Each hunt includes:

- Hypothesis  
- Data sources  
- Queries (Sigma, KQL, Wazuh, Splunk SPL, etc.)  
- Findings and evidence  
- Screenshots  
- MITRE ATT&CK mapping  
- Recommendations  

These hunts demonstrate my ability to proactively search for malicious activity beyond alerts.
